# 简介
* 颜色常用基础类：CMYKColor、HsbColor、ColorHelper；
* 绘图常用基础类：SizeHelper、PointHelper、Padding；
* 图片水印，支持文本水印、透明图片水印；
* 随机图像：验证码生成；
* 常用图像处理方法封装（BitmapHelper）：亮度、对比度、二值化、灰度、细化、轮廓等；
